# Gym Management Website

This project is a modern, responsive Gym Management Website built with React, Node.js, Express, MongoDB, and Razorpay.

## Directory Structure

* `/frontend` - React + Vite + Tailwind CSS frontend application.
* `/backend` - Node.js + Express + MongoDB backend API.
